# Output Guidelines
1. Do NOT write long explanatory intros or summaries.
2. Before performing actions, display a 📋 **Task Overview** checklist using markdown check-boxes.
3. For file updates, output only a brief summary table of modified lines instead of explaining the changes in paragraphs.
4. Keep all responses under 3 paragraphs unless explicitly asked for detailed explanations.
